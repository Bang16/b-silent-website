# B'Silent Florist Website 
 
A beautiful, responsive website for a premium florist business with WhatsApp order integration.

## Live Demo
[**Visit Live Site**](https://bsilentofficial.co.za)

## Project Overview
- **Client:** B'Silent Florist
- **Project Type:** Business Website + E-commerce
- **Status:** Live & Taking Orders
- **Technologies:** HTML5, CSS3, JavaScript, PHP

## Features
- ✅ Responsive Design (Mobile-First)
- ✅ WhatsApp Order Integration
- ✅ Contact Forms with PHP Processing
- ✅ Delivery Management System
- ✅ Professional Branding & UX

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** PHP for form handling
- **Payment:** WhatsApp-based order system
- **Hosting:** domains.co.za

## Key Features Implemented

### E-commerce Functionality
- Shopping cart with localStorage
- Dynamic pricing with delivery fees
- Order summary generation
- WhatsApp order automation

### User Experience
- Mobile-responsive design
- Smooth animations and transitions
- Professional contact forms
- Clear call-to-actions

### Business Solutions
- Custom delivery fee calculator
- Order management workflow
- Customer communication system
- Professional branding

## Responsive Design
